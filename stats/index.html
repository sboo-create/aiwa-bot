Как читать метрики
Ever used — уникальные псевдонимные пользователи с продуктовой активностью за всё время.
Daily … / user — итог периода, делённый на сумму дневных активных пользователей; корректно сравнивает окна 1/7/30 дней.
Request success — запрос считается успешным, если хотя бы одна попытка провайдера завершилась успешно.
Provider attempt errors — технические retry/403/timeouts; один пользовательский запрос может содержать несколько таких попыток.
Observed — событие записано новым точным контуром v2. Reconstructed — безопасно восстановлено из legacy и помечено отдельно.
Стоимость — только вызовы с точным reported/estimated cost. Legacy total tokens без split в точную стоимость не входят.
